# 🎮 단어 복습 게임

아이들의 영어 단어 복습을 위한 웹 게임입니다. 설치 없이 브라우저에서 바로 실행됩니다.

**▶ 바로 실행: https://jhnoh0826.github.io/word-review-game/**

## 주요 기능

- 👧👦 **학생별 관리** — 학생 카드를 추가하고 이름·캐릭터·색상을 자유롭게 설정
- 🎲 **4가지 퀴즈 모드** — 전체 복습 / 뜻 확인 / 스펠링 / 문장 빈칸 채우기
- 📝 **오답 노트** — 틀린 단어가 자동으로 모여서 다시 풀 수 있어요
- ⭐ **점수 & 콤보** — 연속 정답 보너스로 게임처럼 재미있게
- 📋 **간편한 단어 등록** — AI(GPT/제미나이) 프롬프트 자동 생성, 마크다운 표·구글시트 붙여넣기 지원

## 사용법

### 1. 학생 카드 만들기
첫 화면에서 **＋ 새 학생 추가**를 누르고, ✏️ 버튼으로 이름과 캐릭터를 설정하세요.

### 2. 단어 넣기
학생 선택 → **⚙️ 단어 목록 관리** → **＋ 추가**

1. 복습할 단어를 입력하거나, **📷 사진/PDF 선택**으로 교재 사진·PDF를 올리세요
2. 베타 코드를 입력하세요 (안내받은 암호, 한 번 넣으면 기억됩니다)
3. **🤖 AI로 바로 생성** 클릭 → AI가 한글 뜻·영영 풀이·예문까지 자동으로 만들어줍니다
4. 미리보기 확인 → **추가하기** 완료!

구글시트 표 붙여넣기("📋 표 붙여넣기로 추가")와 한 단어씩 직접 입력("✍️ 직접 입력하기")도 지원합니다.

### 3. 게임 시작
원하는 모드를 골라 시작하세요. 문제 수는 한 판에 30문제입니다.

## 데이터 저장 안내

단어 목록·점수·오답 노트는 **사용하는 브라우저의 로컬 저장소(localStorage)** 에만 저장됩니다.

- 사용자끼리 데이터가 섞이지 않아요 (각자 자기 단어로 사용)
- 휴대폰과 PC는 따로 저장돼요
- 브라우저 데이터(사이트 데이터)를 지우면 단어와 기록도 사라지니, 원본 단어는 구글시트 등에 보관해 두세요

## 기술 구성

순수 HTML + CSS + JavaScript로 만든 정적 웹앱입니다. 서버나 데이터베이스가 없으며, `index.html`을 더블클릭해 오프라인으로도 실행할 수 있습니다.
