단어 복습 퀴즈

누구의 단어를 복습할까요?

✏️ 버튼으로 이름과 캐릭터를 바꿀 수 있어요

❓ 이렇게 사용해요

1
학생 카드 만들기

첫 화면에서 ＋ 새 학생 추가 카드를 누르고, 각 카드의 ✏️ 버튼으로 이름과 캐릭터를 바꿀 수 있어요.

2
단어 넣기

학생 선택 → ⚙️ 단어 목록 관리 → ＋ 추가

복습할 단어를 입력하거나, 📷 사진/PDF 선택으로 교재 사진·PDF를 올리세요 (둘 다 해도 돼요) ↓ 베타 코드를 입력하세요 (안내받은 암호 · 한 번 넣으면 기억돼요) ↓ 🤖 AI로 바로 생성 클릭 → 몇 초 뒤 AI가 뜻과 예문까지 만들어줘요 ↓ 미리보기 확인 → 추가하기 완료! 🎉

단어는 세트(폴더)로 나눠 저장할 수 있어요 — 추가할 때 "저장할 세트"에서 날짜별·시험범위별 세트를 고르거나 새로 만드세요. 구글시트 표는 "📋 표 붙여넣기로 추가", 한 단어씩은 "✍️ 직접 입력하기"로도 넣을 수 있어요.

3
퀴즈 시작

세트가 여러 개면 메뉴의 📂 복습 범위에서 오늘 공부할 세트만 골라 풀 수 있어요. 전체 복습 · 뜻 확인 · 스펠링 · 문장 쓰임 중 골라서 시작하세요. 틀린 단어는 오답 노트에 자동으로 모여서 다시 풀 수 있어요.

💡 단어와 점수는 지금 쓰는 브라우저에만 저장돼요. 휴대폰과 PC는 따로 저장되고, 브라우저 데이터를 지우면 함께 사라져요.
