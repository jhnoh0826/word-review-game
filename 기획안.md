# 단어 복습 게임 — 기획안

> 아이들이 공부한 영어 단어를 게임처럼 즐겁게 복습하는 웹 프로그램

---

## 1. 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 단어 복습 게임 (Word Review Game) |
| **한 줄 소개** | 학생이 공부한 단어 리스트로 다양한 테스트 문제를 풀며 복습하는 웹사이트 |

## 2. 목표와 컨셉

### 핵심 목표
- 학생이 **이미 공부한 단어**를 반복 복습해서 장기 기억으로 굳히도록 돕는다.
- 단순 암기가 아니라 **뜻 · 철자 · 문장 속 쓰임**까지 다각도로 확인한다.
- "공부"가 아니라 "게임"으로 느끼게 해서 스스로 반복하게 만든다.

### 설계 원칙
1. **단어 리스트만 바꾸면 끝** — 단어 데이터를 교체/추가하면 모든 문제가 자동 생성된다.
2. **학생별 독립 관리** — Ben과 Lily의 단어, 점수, 오답이 서로 섞이지 않는다.
3. **간단한 구조** — 서버·데이터베이스 없이 파일만으로 동작해서 유지·관리가 쉽다.
4. **틀린 것에 집중** — 틀린 단어를 모아 다시 복습할 수 있게 한다.

---

## 3. 대상 사용자 시나리오

> **상황:** Ben이 이번 주에 학교에서 영어 단어 15개를 배웠다.
>
> 1. 부모/선생님이 그 단어들을 프로그램의 단어 리스트에 입력한다.
> 2. Ben이 프로그램을 열고 자기 캐릭터(🦊)를 선택한다.
> 3. "전체 복습" 모드로 10문제를 푼다. 뜻 맞히기, 철자 쓰기, 문장 빈칸이 섞여서 나온다.
> 4. 틀린 단어 3개는 자동으로 "오답 노트"에 쌓인다.
> 5. 다음 날 다시 들어와 "오답 노트" 모드로 틀렸던 단어만 다시 푼다.
> 6. 점수가 오를수록 최고 점수가 갱신되며 동기부여가 된다.

## 4. 화면 구성 및 흐름

```
[학생 선택]  →  [메뉴]  →  [퀴즈 진행]  →  [결과]
   ↑              ↑___________________________|
   |__________________________________________|
```

### 4.1 학생 선택 화면
- Ben 🦊 / Lily 🐰 카드 중 선택해서 시작.
- 추후 학생을 추가할 수 있는 구조.

### 4.2 메뉴 화면
- 선택한 학생 이름과 **최고 점수(⭐)** 표시.
- 복습 모드 목록에서 원하는 방식 선택 (v1: 5종 → 최종 목표: 11종).
- 오답 노트에 쌓인 단어 개수 표시.
- 단어 리스트가 비어 있으면 안내 문구 노출.

### 4.3 퀴즈 화면
- 상단: 점수 ⭐, 콤보 🔥, 진행바, 문항 번호(예: 3/10).
- 중앙: 문제와 보기/입력창.
- 답하면 즉시 정답/오답 피드백 → "다음" 버튼.

### 4.4 결과 화면
- 맞힌 개수, 획득 점수, 최고 점수 경신 여부.
- 정답률에 따라 다른 축하 메시지(🏆 / 🎉 / 💪 / 📚).
- "다시 하기" / "메뉴로" 버튼.

## 5. 복습 모드 (테스트 유형)

모드는 **현재 구현된 것(v1)**과 **개발 예정**으로 나뉜다.

### 5.1 현재 구현 (v1 — 5종)

| 모드 | 아이콘 | 문제 유형 | 확인하는 능력 |
|------|--------|-----------|---------------|
| **전체 복습** | 🎲 | 아래 퀴즈형 모드가 랜덤으로 섞여서 출제 | 종합 |
| **뜻 확인** | 📖 | ① 단어 → 한글 뜻 4지선다<br>② 영영 풀이 → 단어 4지선다 | 의미 이해 (한글·영영) |
| **스펠링** | ✏️ | ① 뜻 보고 철자 직접 타이핑<br>② 글자 타일 순서 맞추기 | 철자 정확도 |
| **문장 쓰임** | 💬 | 예문의 빈칸에 알맞은 단어 고르기 | 문맥 속 활용 |
| **오답 노트** | 📝 | 이전에 틀린 단어만 모아서 재출제 | 약점 보완 |

> **자동 문제 생성:** 객관식 오답 보기는 같은 학생의 다른 단어에서 자동으로 가져온다. 예문 빈칸은 예문에서 해당 단어를 자동으로 가려서 만든다.

---

### 5.2 개발 예정 (6종)

#### 🃏 3단 변신 플래시카드 (3-Step Flip) — 난이도 ★☆☆☆☆
- **방식:** 카드 앞면에 영어 단어 표시 → 클릭 1회: 영영 풀이 → 클릭 2회: 한국어 뜻 → 클릭 3회: 예문
- **효과:** 빠르게 전체 단어를 훑으며 자기 확인식 복습
- **특징:** 정답/오답 판정 없이 자기 주도로 진행. 모든 단어를 한 사이클에 볼 수 있음

#### 🧩 메모리 매치 (Memory Match) — 난이도 ★★☆☆☆
- **방식:** 카드 전체가 뒤집혀 있음. 한 번에 2장씩 뒤집어 '영어 단어'와 '한국어 뜻'이 일치하는 짝을 찾으면 카드 제거
- **효과:** 단어 뜻 암기 + 집중력 훈련. 모든 짝을 맞추면 완료
- **특징:** 한 판에 단어 6~8쌍으로 제한. CSS 뒤집기 애니메이션 포함

#### 🔍 명탐정 스무고개 (Dictionary Detective) — 난이도 ★★☆☆☆
- **방식:** 영영 풀이의 첫 문장만 먼저 공개 → '힌트 보기' 버튼으로 다음 문장 또는 한국어 뜻을 단계적으로 추가 공개 → 정답 단어 직접 입력
- **점수:** 힌트를 적게 사용할수록 더 높은 점수 (힌트 0개: 30점 / 1개: 20점 / 2개: 10점)
- **효과:** 영어 문장 독해력 향상. 영영 풀이에서 핵심 키워드를 찾는 훈련

#### 🔀 워드 스크램블 개선 (Word Scramble) — 난이도 ★★☆☆☆
- **방식:** 현재 타일 모드를 독립 강화. 뒤섞인 알파벳 버튼을 순서대로 클릭해 단어 완성
- **힌트:** '뜻 보기' / '첫 글자 보기' 버튼 추가
- **효과:** 철자를 눈이 아닌 손으로 직접 조합하며 각인
- **특징:** 기존 스펠링 타일 코드를 확장하여 구현

#### 💬 빈칸 채우기 애니메이션 (Fill-in-the-Blank) — 난이도 ★★★☆☆
- **방식:** 예문 빈칸 채우기 + 정답 선택 시 단어가 빈칸으로 날아가는 애니메이션 추가
- **업그레이드:** 보기 선택 외에 직접 타이핑 모드도 제공 (스펠링+문맥 동시 훈련)
- **효과:** 단어가 실제 문장 속에서 어떻게 쓰이는지 맥락과 함께 익힘

#### 🐉 보스 스테이지 (Boss Stage) — 난이도 ★★★★☆
- **방식:** 화면에 몬스터 등장. 한 단어당 3단계 콤보 공격으로 물리침
  - 1타: 뜻 보고 알맞은 단어 선택 (객관식)
  - 2타: 그 단어의 철자 직접 타이핑
  - 3타: 예문 빈칸에 단어 채워 넣기
- **보상:** 모든 단어(몬스터)를 처치하면 클리어 화면 + 배지/점수 획득
- **효과:** 뜻·철자·문맥을 한 단어에서 연속으로 확인하는 종합 최종 테스트
- **특징:** 공수가 가장 크지만 완성 시 아이들 흥미 유발 효과 가장 높음

## 6. 게임 요소 (동기부여 장치)

- **점수:** 정답마다 기본 10점.
- **콤보 보너스:** 연속 정답이 이어지면 추가 점수(최대 +10점), 3연속부터 🔥 표시.
- **최고 점수:** 학생별로 저장되어 메뉴에 항상 표시 → 자기 기록 갱신 도전.
- **즉각 피드백:** 정답은 초록색·콤보 애니메이션, 오답은 흔들림 효과와 정답 안내.
- **결과 등급:** 정답률에 따라 트로피/박수/응원 메시지로 차등 표시.

### 향후 추가 후보
- 별/코인 적립 및 누적 보상
- 타이머 도전 모드
- 출석/연속 학습일(스트릭) 표시

## 7. 데이터 구조

학생별로 단어 목록을 분리해 관리한다. 단어 하나는 아래 4개 정보로 구성된다.

```javascript
{
  word:       "brave",                                    // 영어 단어
  meaning_ko: "용감한",                                    // 한글 뜻
  meaning_en: "ready to face danger without fear",        // 영영 풀이
  example:    "The brave firefighter ran into the house." // 예문 (단어 포함)
}
```

- 이 4개 필드만 있으면 **모든 모드의 문제가 자동 생성**된다.
- 단어를 추가/삭제/수정하면 복습 범위가 즉시 바뀐다.
- 입력 방식: 현재는 데이터 파일(`js/words.js`)을 직접 수정. (→ 향후 화면 입력 기능 검토)

### 저장되는 진행 데이터 (학생별, 브라우저에 자동 저장)
- `bestScore` : 최고 점수
- `wrongWords` : 틀린 단어 목록(오답 노트). 다시 맞히면 자동 제거.

## 8. 기술 구성

| 영역 | 선택 | 이유 |
|------|------|------|
| **구조** | 순수 HTML + CSS + JavaScript | 서버·DB 불필요, 더블클릭으로 실행 |
| **데이터** | JS 파일(`words.js`) | 비개발자도 형식만 맞추면 수정 가능 |
| **저장** | 브라우저 localStorage | 학생별 점수·오답을 기기에 자동 보관 |
| **배포(선택)** | GitHub Pages 등 정적 호스팅 | 무료, 인터넷으로 어디서나 접속 가능 |

### 파일 구성
```
단어복습게임/
├─ index.html        # 화면 구조 (실행 진입점)
├─ css/
│  └─ styles.css     # 디자인
├─ js/
│  ├─ words.js       # 단어 리스트 (학생별) ← 주로 여기만 수정
│  └─ app.js         # 게임 로직
└─ 기획안.md          # 이 문서
```

## 11. 열린 결정 사항

향후 진행하며 정할 부분:

1. **단어 입력 방식** — 코드 수정 유지 vs. 화면 관리 페이지 추가 (어느 시점에?)
2. **영영 뜻 비중** — 학생 수준에 맞춰 영영 문제 난이도/비중 조절
3. **배포 여부** — 한 대의 PC에서만 쓸지, 인터넷에 올려 어디서나 쓸지
